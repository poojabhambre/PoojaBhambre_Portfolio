Project Details

Speech Emotion Recognition

This project analyzes audio inputs to detect emotions, combining speech processing with emotional AI. It aims to assist industries such as customer service, healthcare, and education by identifying and interpreting human emotions effectively.

Technologies Used:
  • Frontend: HTML, CSS
  • Backend: Python (Flask, Django)
  • Database: SQLite
  • Deep Learning Frameworks: TensorFlow, Keras
  • Audio Processing Libraries: Librosa, NumPy
Use Cases:
  • Customer support: Identifying callers' emotional states to enhance service.
  • Mental health monitoring: Assisting therapists by detecting patients' emotions.
  • Education: Gauging students' engagement during virtual classes.

Crick Base

A comprehensive platform for cricket enthusiasts, Cric Base provides match statistics, player analysis, and team performance tracking, all presented in an intuitive and visually appealing interface.

Technologies Used:
  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Backend: Python (Flask, Django)
  • Database: MySQL or MongoDB
  • APIs: CricAPI (or similar cricket stats APIs)
  • Visualization Libraries: Chart.js, D3.js
Use Cases:
  • Fantasy cricket: Helping users make informed team decisions based on stats.
  • Match analysis: Offering detailed performance trends for teams and players.
  • Fan engagement: Delivering live updates and rankings in real time.
